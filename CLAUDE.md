## Project Overview

RankForged AI is an AI-powered Local SEO SaaS platform designed to help businesses improve their visibility in local search results through automation, optimization, analytics, and AI-driven recommendations.

This project is built using:
- WordPress
- Divi 5
- SEOPress
- Meta Box Lite
- WP All Import
- Dynamic Local SEO Pages
- AI-generated local SEO content
- Structured data/schema
- Subscription-based SaaS architecture

The visual style should reflect:
- Modern SaaS platforms
- Technical SEO dashboards
- Enterprise AI software
- Local search authority

Design inspiration:
- SEMrush
- BrightLocal
- Ahrefs
- Surfer SEO
